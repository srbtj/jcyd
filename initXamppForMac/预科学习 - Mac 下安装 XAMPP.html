一、 下载XAMPP并安装
    https://www.apachefriends.org/download.html  选择PHP7.3.2
    安装注意事项：
      1. Mac默认安装至 /Application/Xampp 目录下
      2. 选择
            XAMPP Core file 核心文件，包含apache, mysql文件等
            Xampp Developer Files 开发依赖文件
      3. 成功后 重启apache服务

二、 设置启动的PHP, mac系统自带安装 php, 一般与 XAMPP 自还的版本号不一致
      php -v : 查看当前系统默认的php
      在命令行中修改 .bash_profile文件， 更改php
      export PATH=/Applications/XAMPP/xamppfiles/bin:$PATH

      
 三、 设置Apache的 VirtualHost  (当前用户目录 /Users/tougou)
        1. 进入 XAMPP 安装目录  /Applications/XAMPP/etc
        2. vi httpd.conf   取消注释
            
        3. cd  /Applications/XAMPP/etc/extra
            

      4. 重启 apache 验证

      5. 启动所有
         
  
启动前，若本地安装了mysql ,先关闭本地mysql服务， 避免冲突
